  • PING - represents the amount of time it takes for a communications packet to get from your computer to the server and back again.
  • GWTL - represents GetWorkTrafficLatency which determines how fast the server responds to your miners request for mining work.
  • In short, you want the lowest 'Ping' and a low 'GWTL' when selecting a public p2pool node.
